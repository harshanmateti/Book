About Us

The project is built using Node.js, MongoDB, and HTML, designed to manage a collection of books. It allows users to store book content in a database, enabling them to add new books with detailed information such as title, author, and content. The backend is powered by Node.js, which interacts with MongoDB for data storage and retrieval. The frontend, built with HTML, provides a simple and user-friendly interface for displaying and managing the books. Users can view the content of each book and have the option to delete any book they no longer need, which will remove the associated data from the MongoDB database. This project demonstrates how a full-stack application can be developed to manage book data efficiently.
